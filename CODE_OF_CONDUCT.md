# Code of Conduct

We want this project to be a welcoming place for thoughtful collaboration.

Be respectful, assume good intent, give actionable feedback, and make room for different experience levels. Harassment, discrimination, threats, doxxing, or publication of another person's private data are not acceptable.

Maintainers may edit or remove content and restrict participation when needed to protect the community. Report conduct concerns privately through the maintainers' GitHub contact channel.
