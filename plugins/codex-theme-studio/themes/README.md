# Bundled themes

The collection contains eighteen bundled backgrounds. The first six abstract themes were created
with ChatGPT on 2026-07-17; six character-led themes were created with OpenAI ImageGen on
2026-07-18; and six “万妖图录” themes were supplied by the project owner and explicitly authorized
for public redistribution under CC BY 4.0 on 2026-07-19. Each supplied source is a 1672×941 PNG.
The runtime keeps the original pixels and uses `cover`; it does not claim a synthetic 4K upscale.

The featured first theme in the “万妖图录” collection is `wan-yao-longyuan-lingji`. Installation
does not silently apply it; the user still chooses preview or apply.
