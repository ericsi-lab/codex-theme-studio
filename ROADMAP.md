# Roadmap

## v0.1.x — macOS hardening

- Validate `doctor → preview → verify → restore` on a signature-valid official ChatGPT build.
- Measure idle CPU, P95 CPU, apply RSS and retained RSS after 30 switches on the real renderer.
- Complete home/task, light/dark, narrow/wide, modal, streaming and reduced-motion visual checks.
- Replace proof placeholders with privacy-safe real captures.
- Publish the repository and first Release after the GitHub owner is confirmed.
- Record licensed narration/BGM, render the launch variants, and validate them with `ffprobe`.

## v0.2 — Theme authoring

- Visual focus-point and safe-area editor.
- Local palette preview and contrast guidance.
- Exportable theme package validation report.

## Future

- Windows support after a platform-specific process, signature, port, path, restore, and uninstall security review. It is not supported by v0.1.0.
