# Theme Studio for Codex

This archive contains the installable Theme Studio for Codex Plugin for macOS.
It is an unofficial community project and is not affiliated with or endorsed by OpenAI.

## Install

Add this extracted directory as a Codex Plugin marketplace, install **Theme Studio for Codex**,
then open a new task and say `安装主题` or `install themes`.

Detailed guides:

- [中文安装说明](docs/INSTALL.zh-CN.md)
- [English installation guide](docs/INSTALL.md)
- [Theme format](docs/THEME_FORMAT.md)
- [Upgrade compatibility](docs/UPGRADE_COMPATIBILITY.md)

Source, screenshots, issues, and release checks are maintained at
[ericsi-lab/codex-theme-studio](https://github.com/ericsi-lab/codex-theme-studio).

To remove every injected decoration and return to the official appearance, open a new task and say
`恢复 Codex 默认界面` or `restore the original Codex appearance`.
