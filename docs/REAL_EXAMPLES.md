# 真实应用效果

以下图片来自通过签名校验的官方 ChatGPT 桌面端 renderer，不是概念图，也不是把背景图
贴进自制 UI。Theme Studio 通过仅限回环地址的 CDP 应用主题，并在每张图片写盘前验证
主题 ID、Adapter、页面类型、交互结构和演示模式。

- 截图版本：ChatGPT `26.715.31925`（build `5551`）
- Adapter：`12`
- 视图：完整新建任务窗口、官方设置窗口
- 隐私：演示模式只隐藏头像、用户名和邮箱等账户身份；项目侧栏、项目名称、工作区标签、
  输入框和应用结构均保持真实，不裁剪、不遮挡
- 不包含任务工作页、对话正文、终端输出或绝对本地路径

设计预览稿位于各主题目录中的 `preview-light.jpg` / `preview-dark.jpg`；它们用于主题设计
评估，不应与本页真实应用截图混淆。机器可读的捕获记录见
[`examples/manifest.json`](examples/manifest.json)。
